# PyUnzip :unlock:

PyUnzip adalah sebuah program Python3 yang dirancang untuk meng-crack kata sandi file Zip dengan menggunakan dua metode serangan utama: Brute Force Attack dan Dictionary Attack. Brute Force Attack memungkinkan pengguna untuk mengatur panjang dan jenis karakter yang digunakan dalam pencarian kata sandi, sedangkan Dictionary Attack menggunakan sebuah file Wordlist untuk mencoba setiap kata sandi yang ada. 

 > **Peringatan:** Program ini hanya untuk tujuan Edukasi dan Pembelajaran. Jangan gunakan program ini untuk tujuan ilegal!

## Instalasi

### Instalasi di Android (Termux)

> Untuk menginstal PyUnzip di Android (Termux) ketikkan perintah berikut:

```
pkg update
pkg install python3
pkg install git
git clone https://github.com/bgropay/PyUnzip/
cd PyUnzip
python3 -m venv modules
source modules/bin/activate
pip3 install -r requirements.txt
```

### Instalasi di Linux (Debian/Ubuntu)

> Pastikan Anda sudah dalam mode super user (root) sebelum menginstal PyUnzip. Jika sudah, ketikkan perintah berikut:

```
apt-get update
apt-get install python3
apt-get install python3-pip
apt-get install git
git clone https://github.com/bgropay/PyUnzip/
cd PyUnzip
python3 -m venv modules
source modules/bin/activate
pip3 install -r requirements.txt
```

### Instalasi di Windows 

> Untuk menginstal PyUnzip di Windows, pastikan Python3 sudah terinstal. Jika sudah, unduh Source Code berikut:

[Source Code](https://github.com/bgropay/PyUnzip/archive/refs/heads/main.zip)
