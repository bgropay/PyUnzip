# PyUnzip

PyUnzip adalah program yang dirancang oleh BgRopay untuk melakukan cracking terhadap kata sandi file ZIP. PyUnzip menggunakan tiga teknik serangan yaitu Brute Force Attack, Dictionary Attack, dan Combinator Attack. Teknik serangan Bruteforce Attack mencoba semua kombinasi karakter sesuai kriteria yang diberikan hingga menemukan kata sandi yang benar atau semua kemungkinan telah dicoba. Teknik serangan Dictionary Attack mencoba setiap kata dalam wordlist sebagai kata sandi untuk file Zip. Teknik serangan Combinator Attack menggabungkan setiap kata dari kedua wordlist dan mencoba kombinasi tersebut sebagai kata sandi untuk file Zip.
